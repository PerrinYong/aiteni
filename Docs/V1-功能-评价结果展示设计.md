
# 完整评语内容结构
## 结构说明
完整评语（即你 Demo 里的长版本）统一由以下几部分组成：

1. 总体等级说明
2. 各维度得分列表
3. 主要优势（Top 优势维度）
4. 提升重点（优先补短维度）
5. 详细维度评语（逐维度展开）
6. 可选附加提示（练习重点 / 使用建议）

### 1. 总体等级说明

**目的**：一眼告知用户自己属于哪个等级段、是什么类型的选手。

**包含内容：**

* `总体等级`: 如 `NTRP 3.0`
* `等级标签`: 如 `发展中业余选手`（由 `roundedLevel → levelLabel` 映射而来）
* `原始得分`: 可选显示，如：`原始得分: 3.00`

**文案示例：**

```text
整体来看，你当前的综合水平约为 NTRP 3.0（发展中业余选手）。
```

如需要，还可以在完整版的顶部做一个「结果标题区」，包括：

```text
🎾 总体等级: NTRP 3.0 (发展中业余选手)
原始得分: 3.00
```

---

### 2. 各维度得分列表

**目的**：展示用户在每一个能力维度上的评分，为后面雷达图与维度评语提供数据。

**维度列表（当前版本）：**

* 底线综合（稳定性+深度）
* 正手
* 反手
* 发球
* 接发球
* 网前与高压
* 步伐与场地覆盖
* 战术与心理
* 实战成绩
* 训练背景 / 频率

**展示形式（文本形式示例）：**

```text
📊 各维度得分:
  • 底线综合（稳定性+深度）: 2.5
  • 正手: 3.0
  • 反手: 3.0
  • 发球: 2.8
  • 接发球: 3.0
  • 网前与高压: 3.5
  • 步伐与场地覆盖: 3.0
  • 战术与心理: 3.5
  • 实战成绩: 4.0
  • 训练背景 / 频率: 3.5
```

在 UI 中，这部分会对应雷达图和条形图的数据源。

---

### 3. 主要优势（优势维度总结）

**目的**：用简练语言告诉用户「我哪里强」，增强自我认知和自信。

**来源逻辑：**

* 从 `dimensionScores` 中选出得分最高的 2–3 个维度；
* 同时参考它们相对总体等级的差值（高于 `roundedLevel` 一定阈值视为优势）。

**展示内容：**

* 优势维度名称列表（简略版使用）
* 每个优势维度的一句简短描述（详细版使用）

**文案示例：**

```text
💪 主要优势: 实战成绩, 网前与高压, 战术与心理
```

详细版中：

```text
你的主要优势在：
- 实战成绩（约 4.0 级）：你对俱乐部高手的对抗成绩不错，说明自身水平在同圈层已经具备明显竞争力，可以多参加正式比赛或级别更高的对手检验自己。
- 网前与高压（约 3.5 级）：网前已经具备一定威胁，可以针对低球截击、反手截击和连续截击做专项训练，让你在合适机会时更有把握通过一两拍结束这一分。
- 战术与心理（约 3.5 级）：战术意识较好，能根据对手特点调整打法，可以结合简单的比赛复盘，多思考“对某类选手什么策略成功率最高”，持续优化你的比赛模板。
```

---

### 4. 提升重点（优先补短维度）

**目的**：给出「接下来先练什么」的清晰指引。

**来源逻辑：**

* 从 `dimensionScores` 中选出得分最低的 2–3 个维度；
* 或用「总等级 - 维度得分」的差值做排序，差值最大的为优先补短项。

**展示内容：**

* 提升重点维度名称列表（简略版）
* 每个短板维度的简短改进建议（详细版）

**文案示例：**

```text
🎯 提升重点: 底线综合（稳定性+深度）, 发球, 步伐与场地覆盖
```

详细版中：

```text
当前最值得优先提升的环节是：
- 底线综合（稳定性+深度）（约 2.5 级）：底线相持已经有一定稳定性，但深度和变化略显不足，可以通过刻意练习深球和落点变化来增加威胁，而不是只把球挂在中场。
- 发球（约 2.8 级）：发球已经能承担“开始一分”的任务，可以逐步区分一发和二发，在保证二发稳定的前提下提升一发威胁，增加落点与旋转变化。
- 步伐与场地覆盖（约 3.0 级）：步伐基础尚可，可以多练习“左右调动 + 短球/高吊”组合，让你在多变球路下仍能打出稳定回球，而不至于被一两拍突然变化拉垮。
如果你只想抓重点，建议优先在上述 2～3 个方向投入练习时间。
```

### 5. 详细维度评语（逐维度展开）

**目的**：提供每个维度的具体诊断和练习建议，支撑长期训练计划。

**格式统一：**

```text
【维度名称（约 X.X 级）】
维度评语长句（基础评语 + 优势/短板补充）
```

**评语组成：**

1. 「基础评语」：根据该维度得分落在哪个区间（如 <2.5、2.5–3.5、>3.5）选取对应模板；
2. 「相对评语」：根据该维度相对总体等级的高低，补一句「优势 / 短板 / 中性」点评。

**示例：**

```text
【底线综合（稳定性+深度）（约 2.5 级）】
底线相持已经有一定稳定性，但深度和变化略显不足，可以通过刻意练习深球和落点变化来增加威胁，而不是只把球挂在中场。这一项相对是短板，会在比赛中拖慢整体上限，建议作为近期重点练习方向。

【正手（约 3.0 级）】
正手已具备一定稳定性和方向控制，可以在保证成功率的前提下，增加深度和上旋量，让正手逐渐从“安全球”变成“有威胁的球”。这一项与整体水平大体一致，可以在保持稳定的基础上，循序渐进地提高质量。
```

所有 10 个维度按此格式逐条列出。

### 6. 可选附加提示

可在末尾加一小段「练习建议总览」或「如何使用结果」：

```text
如果你每周有 2～3 次打球时间，建议在保证正常对抗的基础上，每周抽出 1 次做“针对短板的专项练习”，例如专门练发球+第一拍、底线深度控制等。定期（例如每 3 个月）重新做一次评估，可以观察自己在各个维度上的变化趋势。
```

# 二、评价结果展示卡片设计

前端展示设计为两层：

* **简略版卡片**：在列表页 / 结果概览中使用；
* **详细版卡片**：点击后进入详情页，展示完整话术。

## 简略版卡片（概览卡）

**适用场景：**

* 用户刚做完评估后的「结果首页」；
* 历史评估记录的列表项；
* 需要在小空间内给出「一眼看懂」的信息。

**包含元素：**

1. 总体等级（数字 + 标签）
2. 各维度得分 – 雷达图（摘要）
3. 主要优势 – 简述（2–3 个维度名）
4. 提升重点 – 简述（2–3 个维度名）
5. 查看详情 / 展开按钮

#### 1.1 信息结构建议

```text
[卡片顶部]
🎾 NTRP 3.0
发展中业余选手

[中部]
📊 能力雷达图（底线/正手/反手/发球/网前/步伐/战术/实战 ...）

[底部]
💪 主要优势：实战成绩 / 网前与高压 / 战术与心理
🎯 提升重点：底线综合 / 发球 / 步伐与场地覆盖

[按钮]
[查看完整评语]
```

* 雷达图：只用 6–8 个核心维度，保证手机上标签可读；
* 优势 / 提升重点：只显示维度名字，不展开长文案。

#### 1.2 文案风格建议

* 尽量短句、标签式，避免挤满一屏文字；
* 例如：

```text
💪 优势：网前收分、比赛经验、战术意识
🎯 补短：底线稳定性、发球威胁、场地覆盖
```

## 详细版卡片设计

### 整体结构

详细版卡片建议采用如下结构顺序：

1. 结果头部（总体等级 + 简短一句总结）
2. 总体摘要段落
3. 优势维度 – 展开描述
4. 提升重点 – 展开描述
5. 各维度得分与评语（逐维度展开）
6. 结尾建议（可选）

**目的**：用一眼就能看懂的方式告诉用户自己的等级。

**包含内容：**

* 等级：`NTRP 3.0`
* 等级标签：`发展中业余选手`
* 原始得分（可选）：`3.00`

### 总体摘要段落

**示例：**

```text
整体来看，你当前的综合水平约为 NTRP 3.0（发展中业余选手）。
在同水平玩家中，你已经具备一定的实战竞争力，尤其在比赛经验、网前处理和战术意识上表现较好。

```

（可以由评语引擎根据总等级 + 优势/短板自动生成模板化摘要。）

* 标题行：例如 `💪 你的主要优势：`
* 下方用 **列表形式** 展开每个优势维度：

  * 维度名称 + 等级（括号）
  * 1～2 句说明「现状表现」
  * 1～2 句说明「如何继续放大优势 / 作为得分手段」

#### 示例（基于你的 Demo）

```text
💪 你的主要优势：

  网前已经具备一定威胁，可以通过截击、高压等手段在合适机会时结束这一分。

- 战术与心理（约 3.5 级）：

```

### 提升重点 – 展开描述

**目的**：告诉用户「最应该优先练的 2～3 个点是什么」，**每个点要说明现状问题 + 怎么练**。

#### 内容结构

* 标题行：例如 `🎯 当前最值得优先提升的环节：`
* 下方列表形式展开每个短板维度：

  * 维度名称 + 等级（括号）
  * 1 句描述现状问题 / 短板
  * 1～2 句给出具体练习方向
* 再补一段「如果只抓重点」的总结句。

#### 示例（基于你的 Demo）

```text
🎯 当前最值得优先提升的环节是：

- 底线综合（稳定性+深度）（约 2.5 级）：
  底线相持已经有一定稳定性，但深度和变化略显不足，容易把球挂在中场给对方制造机会。
  建议刻意练习深球和落点变化，例如多做“对角深球 + 直线变线”的对打练习，让你的底线球既稳又有威胁。

- 发球（约 2.8 级）：
  发球已经能承担“开始一分”的任务，但区分度不够，威胁和稳定性尚未同时在线。
  可以逐步区分一发和二发，在保证二发稳定的前提下，提升一发力量和落点变化，并加入一定旋转。

- 步伐与场地覆盖（约 3.0 级）：
  步伐基础尚可，但在被左右调动或应对突然短球、高吊时，衔接还有欠缺。
  建议多练习“左右调动 + 短球/高吊”组合，让你在多变球路下仍能打出稳定回球，而不至于被一两拍突然变化拉垮。

如果你只想抓重点，建议优先在上述 2～3 个方向投入练习时间。
```
